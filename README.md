# Traffic Control & CRM Analytics Dashboard

A comprehensive dashboard for tracking cold traffic performance from Facebook, YouTube, and TikTok ads. Built specifically for info-business (online course sales) with complete sales funnel visibility and CRM integration.

## Features

- **Creative Performance Tracking** - Individual ad metrics and ROI
- **Sales Funnel Analysis** - 7-stage conversion tracking
- **Ad Manager Performance** - Team productivity metrics
- **Platform Comparison** - Facebook, YouTube, TikTok side-by-side
- **Trend Analysis** - Historical data and growth tracking

## Quick Start

1. Install dependencies:
   ```
   npm install
   ```

2. Start development server:
   ```
   npm run dev
   ```

3. Open browser:
   ```
   http://localhost:5020/admin/traffic-control
   ```
